Step-by-Step Setup

  1. Generate an API Key
    Go to Keys page, enter your device name (e.g., "Living Room Light"), and click "Generate API Key". Copy the generated key.
  2. Open Your Device Settings
    Navigate to the device configuration or "Add New Device" screen in your device's app.
  3. Enter Device Name
    Give your device a friendly name (e.g., "Living Room Light", "Bedroom TV")
  4. Enter WebSocket URL
    Copy the WebSocket URL shown above and paste it into the "WebSocket URL" field.
  5. Enter API Key
    Paste the API key you generated in step 1 into the "API Key" field.
  6. Set API Key Header Name
    Enter X-API-Key in the "API Key Header Name" field (if not already set).
  7. Save and Connect
    Click "Add Device" or "Save" to complete the setup. Your device should now be connected!

Troubleshooting

  • Connection Failed: Make sure the server is running and accessible from your device's network
  • Authentication Error: Verify your API key is correct and hasn't been deleted
  • Wrong URL: If your device is on a different machine, use your server's IP address instead of localhost
  • Firewall Issues: Ensure port 1212 is open in your firewall
